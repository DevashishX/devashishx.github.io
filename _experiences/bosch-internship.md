Developed and delivered an analysis tool for embedded driver-assistance systems on ARM platforms to four customer projects.

- Reduced memory leaks across two internal C++ libraries by operationalizing the tool in Conan, CMake, and Python build workflows.
- Owned the full delivery lifecycle from requirements engineering and architecture to implementation, production maintenance, and user support.
- Implemented 12 Qt-based reference projects for live customer demos and reusable C++ middleware integration patterns.
- Integrated tooling and requirement checks into GitHub workflows and Confluence for all pull requests.
- Contributed in agile delivery processes under ISO-26262 and ASPICE constraints with JIRA-based traceability.

Relevant skills: Embedded development, C++, Python, Qt, Conan, CMake, Git, GitHub Actions, Linux, QNX, Bash, JIRA, Confluence, Agile, ISO-26262, ASPICE.
